Compressed Zipped Folders(.zip)

These files typically contain a multitude of files and folders compressed in to a single file ending in the ".zip" extension in its name. Zipped Folders are great for Stand Alone Executables as these types of programs don't need to be installed to call system libraries this also makes them ideal for programs you wish to run from a usb drive.

Pros

  • Quick Setup.
  • Great for Stand Alone Executables.
  • Software can still call funtions from pre-installed libraries.

Cons

  • AntiVirus software may call a false positive on the software.
  • Not Recommended for Powerful Programs
  • If the software requires additional libraries to be installed, this can be a nightmare.

Installation Packages(.exe)

These files are considered to be official setup wizards. They usually install the program as well as the required programs onto the local device. This makes them great for heavy duty programs that make large amounts of calls from multiple libraries. Not only that but it ensures that the files programs and libraries needed to run said files are properly installed secure location while creating shortcuts for easy access after said installation.

Pros

  • Secure Installation makes it less likely to get a false positive returned by your AV Software.
  • Great for Powerful Software.

Cons

  • Longer Setup Time.
  • Installed software is not as portable.
  • Access prompts may appear.
